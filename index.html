Hur lägger jag till en app?
Kopiera ett app-card-block i HTML-koden och ändra:
href → filnamnet på din HTML-app
data-category → verktyg / träning / övrigt
Emoji, titel och beskrivning efter smak.

Ladda sedan upp alla filer till GitHub och aktivera Pages.
1. Skapa GitHub-repo 2. Ladda upp index.html + dina app-filer 3. Settings → Pages → Deploy from main 4. Dela länken! 🚀
